How to Run SMP Admn Stats Application

Important: Do NOT open index.html directly in the browser. It will show "Error Loading Data" due to security restrictions.

✅ Method 1: Use the Start Server Batch File (Easiest)

  1. Double-click on start-server.bat in this folder
  2. A command prompt window will open and start the server
  3. Open your browser and go to: http://localhost:8000
  4. The application will load with all data
  5. Keep the command prompt window open while using the app

Method 2: Manual Command Prompt

  1. Open Command Prompt
  2. Navigate to this folder: cd "path\to\this\folder"
  3. Run: python -m http.server 8000
  4. Open browser and go to: http://localhost:8000

Method 3: VS Code Live Server

  1. Install VS Code if you don't have it
  2. Install the "Live Server" extension
  3. Open this folder in VS Code
  4. Right-click on index.html → "Open with Live Server"

Why does this happen?

Modern browsers block local file access for security reasons when you open HTML files directly (file:// protocol). Running a local server solves this by serving files over HTTP protocol.

Troubleshooting

  • Make sure Python is installed on your system
  • If port 8000 is busy, try: python -m http.server 8080
  • Ensure all CSV files are in the same folder as index.html
